📸 图像处理流程

Fashion MNIST 衣服图像

28×28 像素灰度图像

1️⃣ 随机旋转

输入图像随机旋转 ±10度，增加数据多样性。

transforms.RandomRotation(10)

2️⃣ 随机平移

图像在水平和竖直方向随机平移 ±10%。

RandomAffine(degrees=0, translate=(0.1, 0.1))

3️⃣ 亮度调整

随机调整图像亮度 ±20%，模拟不同光照条件。

ColorJitter(brightness=0.2)

4️⃣ 张量转换

图像转换为 PyTorch 张量，像素值范围 [0, 1]。

transforms.ToTensor()

5️⃣ 标准化

使用均值(0.1307)和标准差(0.3081)进行标准化，加快收敛、提升稳定性。

Normalize(mean=(0.1307,), std=(0.3081,))

✅ 规范的网络层顺序

卷积块： Conv → BatchNorm → ReLU → MaxPool → Dropout
全连接块： FC → BatchNorm → ReLU → Dropout（最后一层无Dropout）
输出层： FC → LogSoftmax（无激活、无Dropout）
